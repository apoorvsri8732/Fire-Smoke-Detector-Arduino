                                    Fire Smoke Guard: An Intelligent Fire and Smoke Detection Solution using Arduino Uno

Problem Statement :-
To design, develop, and implement a reliable, low-cost, and automated system capable of detecting the early signs of fire (smoke and flame) and providing an immediate audible and visual alarm to prevent casualties and property damage.

Primary Objectives:
1.	To interface an MQ-2 smoke sensor and an IR flame sensor with the Arduino Uno microcontroller.
2.	To read and process the analog and digital signals from these sensors.
3.	To program the Arduino to make logical decisions based on sensor data.
4.	To activate output devices (Buzzer) when a fire hazard is detected.
5.	To display the system status and sensor values in real-time on an LCD screen.

Secondary Objectives :-
1.	To assemble the complete circuit on a breadboard.
2.	To test the system under controlled conditions to verify its functionality.
3.	To document the entire process, including limitations and potential future improvements.

Components used:
Arduino Uno 
MQ-2 Smoke/Gas Sensor 
Flame Sensor (IR Receiver)                                  
16x2 LCD Display 
Buzzer 
Breadboard & Jumper Wires 
Power Supply
